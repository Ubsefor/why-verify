# Task 4

## Build

### Build:

```sh
make all
```

### Build with coverage:

```sh
make cov
```

## Run:

### Run:

```sh
make run
```

### Run with coverage:

```sh
make run-cov
```

### Run with valgrind:

```sh
make valgrind
```

### Valgrind with *--leak-check=full*:

```sh
make leak
```

## Coverage:

### Run lcov and collect coverage:

```sh
make gcov
```

## Clean:

```sh
make clean
```
